# 使用分级就绪与受监督 Windows 验收

项目分为 `core_validated`、`windows_validated` 和 `ready`：Mac 先验证协议、Schema、文件库、工作流和模拟 Blog API；目标 Windows 主机及明确声明的 Agent 运行时必须通过受监督 Computer Use 套件；只有改写规范、默认提示词和真实 Blog API 适配完成，并在安全草稿目标跑通端到端流程后才达到 `ready`。任何较低阶段都必须列出缺失条件，不得宣称完整可用。
