# 10 — 适配真实 Blog API

**What to build:** 在同事提供正式接口文档和安全测试环境后，让现有受限交付边界适配真实 Blog 服务的字段、认证、发布目标映射、图片上传、幂等语义和响应模型，并用一个安全发布目标创建可在 Blog 管理端查看的真实发布草稿。

**Blocked by:** 06 — 构建受限且幂等的草稿交付边界

**Status:** ready-for-agent

- [ ] 外部前置条件已满足：已取得正式 API 文档、测试凭据和安全发布目标；在此之前不猜测接口且本票据不得宣称完成。
- [ ] 适配器按真实契约验证请求和响应，完成发布目标映射、必要图片上传、稳定幂等处理及外部草稿标识持久化。
- [ ] 凭据只从运行时环境或 Windows 凭据存储取得，配置仅保存非敏感引用，任务、日志、报告、提示和 Git 中不出现密钥。
- [ ] 对真实服务的超时、未知结果、重复重试和部分图片上传失败均保持幂等且可恢复。
- [ ] 真实客户端的能力允许列表经测试确认不包含公开发布、删除、用户管理、部署或其他管理操作，即使测试凭据本身权限更大。
- [ ] 使用安全发布目标成功创建一篇待人工批准的真实发布草稿，并保留足够证据验证外部结果。
